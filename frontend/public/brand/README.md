# Brand assets

Place the Aditya Birla Capital — Life Insurance logo here as:

    abc-life-insurance.png

The sidebar masthead loads it automatically. If the file is missing, the app
renders a labelled placeholder of a similar aspect ratio (see
src/components/brand/Logo.tsx) so nothing breaks in the demo. The logo is never
recoloured; in dark mode it sits on a small light chip.
